== README

XML feed генератор для Joomla Virtuemart отдельным файлом в формате Yandex

Установка:

* Скопировать в корень сайта

* Обратиться к файлу по адресу с добавочным get параметром http://site.ru/xml_feed.php?code=k59fkj29gja

* Не тратить больше на это время


Тестировалось на:
*  Joomla 2.5
* Virtuemart 2.0.26 RU
